# ruiji_take_out
一个外卖小项目 后端技术 采用springboot + mybatis-plus, 采用mysql数据库
对于项目的优化

后台登录网址
http://localhost:8080/backend/page/login/login.html

前台登录网址
http://localhost:8080/front/page/login.html
